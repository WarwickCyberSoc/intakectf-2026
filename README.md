# IntakeCTF 2026

Challenges files for IntakeCTF 2026. IntakeCTF is Warwick CyberSoc's annual beginner CTF, mainly aimed at the incoming undergraduate and postgraduate cohorts.

Challenges are stored as sub-folders of their respective category, e.g. pwn (binary exploitation) or web (web exploitation).

All challenges include a full writeup and all the files you'd need to host and try them yourself. It's a great resource for anybody looking to get into CTFs!
